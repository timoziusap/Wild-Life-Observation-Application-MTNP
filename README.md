Wild Life Observation Application

Dieses Repository enthält die Wild Life Observation Application, welche im Rahmen einer bewerteten Teamaufgabe im Fach Informationssystementwicklung an der Hochschule Aalen unter der Leitung von Prof. Dr. Marc Fernandes  entwickelt wurde.

Die Anwendung dient der Erfassung, Verwaltung und Visualisierung von Wildtier-Beobachtungsdaten verschiedener Gattungen an unterschiedlichen Orten. Das Projekt teilt sich in eine robuste REST-API (Backend) und einen modernen Webclient (Frontend) auf.
